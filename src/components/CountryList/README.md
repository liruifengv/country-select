# BaseInput
---
CountryList 城市列表组件

### 基本用法

```jsx
<CountryList data={countries} height={600} onSelect={handleSelect}/>
```

###  Attributes
| 参数        | 说明           | 类型  | 可选值 | 默认值 |
| ------------- |:-------------:| :-----|:-----|:-----|
| data   | 数据 |    Arrary | ——— | [] |
| height   | 高度 |  Number | ——— | ——— |
| onSelect   | 选择时间 |   func | ——— | ——— |
